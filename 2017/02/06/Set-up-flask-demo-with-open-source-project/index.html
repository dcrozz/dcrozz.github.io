Set up flask demo with open source project

2月 6, 2017

今天的工作，老板终于要让我做CI了好感动。

###Clone the project and set up virtual environment

  • Choose a workspace to place the project

  • Set the virtual environment

Database connection

Connect mysql with python
  • pip install pymysql
  • Modify config.py
    SQLALCHEMY_DATABASE_URI = 'mysql+pymysql:///username:password@localhost/databasename'
  • run $ python db_create.py
Add user

  • Add user admin whose password is admin

###Set the website can be visited from other hosts

  • Modify run.py from app.run(debug=debug) to app.run(host='0.0.0.0',debug=debug)

Run the server

  • Under the path gitlab/microblog, run python run.py
